# Contact List App

Aplicación web para gestionar contactos de forma sencilla, desarrollada con HTML, CSS y JavaScript.

---

## Demo

https://contact-list-js.vercel.app/

---

## Descripción

Esta aplicación permite crear, visualizar, editar y eliminar contactos desde una interfaz intuitiva.  
Es un proyecto frontend puro, enfocado en la manipulación del DOM y manejo de eventos dinámicos, sin backend ni base de datos.

---

## Funcionalidades

- Crear nuevos contactos  
- Visualizar lista de contactos  
- Editar información de contactos  
- Eliminar contactos  
- Buscar contactos en tiempo real  
- Interfaz responsive y moderna  

---

## Tecnologías

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
